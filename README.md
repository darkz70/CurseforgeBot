# CurseForge → Telegram бот (на GitHub Actions)

Раз в день присылает в Telegram отчёт: сколько скачиваний за сегодня
по каждому вашему проекту на CurseForge + накопленный доход
(доход вводится вручную, т.к. публичного API для выплат у CurseForge нет).

Работает полностью на GitHub — без сервера, бесплатно (GitHub Actions).

## Что нужно сделать

### 1. Создать Telegram-бота
1. Напишите в Telegram **@BotFather** → `/newbot` → следуйте инструкциям.
2. Получите **токен бота** (вид `123456:ABC-DEF...`).
3. Напишите своему боту что угодно (`/start`), затем откройте в браузере:
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
   и найдите `"chat":{"id": ВАШ_ID, ...}` — это ваш **chat_id**.
   (Если хотите слать в группу — добавьте бота в группу и возьмите id группы оттуда же.)

### 2. Получить API-ключ CurseForge
1. Зарегистрируйтесь на https://console.curseforge.com/
2. Создайте API-ключ (раздел API Keys).
3. Узнайте **ID ваших проектов**: откройте страницу проекта на curseforge.com,
   ID указан в правой колонке "About Project" (поле "Project ID"),
   либо через сам API.

### 3. Загрузить этот проект на GitHub
1. Создайте новый репозиторий на GitHub.
2. Загрузите туда все файлы из этой папки (через git или "Add file → Upload files").

### 4. Настроить секреты репозитория
В репозитории: **Settings → Secrets and variables → Actions → New repository secret**

Добавьте:
| Имя | Значение |
|---|---|
| `CURSEFORGE_API_KEY` | ваш API-ключ CurseForge |
| `TELEGRAM_BOT_TOKEN` | токен бота из BotFather |
| `TELEGRAM_CHAT_ID` | ваш chat_id |

### 5. Указать свои проекты
Откройте `config.json` и замените `id` и `name` на свои проекты:
```json
{
  "projects": [
    { "id": 123456, "name": "Название проекта" }
  ]
}
```
Либо, без редактирования файла — задайте переменную репозитория
(**Settings → Secrets and variables → Actions → Variables**)
`CURSEFORGE_PROJECT_IDS` со значением вида `123456,234567`.

### 6. Проверить
Перейдите в **Actions → Daily CurseForge report → Run workflow**,
чтобы запустить отчёт вручную и убедиться, что сообщение пришло в Telegram.

После этого бот будет сам присылать отчёт каждый день в 09:00 МСК
(время можно поменять в `.github/workflows/daily_report.yml`, поле `cron`).

## Как записывать доход
Доход CurseForge не отдаёт по API — вводите его руками:

**Actions → Add money → Run workflow** → укажите сумму и (опционально) комментарий.
Бот сразу подтвердит запись в Telegram, а в следующем ежедневном отчёте
эта сумма попадёт в раздел "💰 Доход".

## Структура проекта
```
.github/workflows/daily_report.yml   — ежедневный запуск отчёта
.github/workflows/add_money.yml      — ручная запись дохода
scripts/daily_report.py              — логика отчёта (скачивания + доход)
scripts/add_money.py                 — логика записи дохода
config.json                          — список проектов CurseForge
data/history.json                    — история скачиваний по дням (авто)
data/money.json                      — история дохода по дням (авто)
```

## Если хочется интерактивного бота (команды /stats, /money и т.д. в любой момент)
GitHub Actions не умеет работать 24/7 и слушать сообщения — только по расписанию
или вручную. Для бота, который отвечает на команды сразу, понадобится
постоянно работающий процесс (например, бесплатный тариф Railway / Render / VPS).
Скажите, если такой вариант нужен — соберу отдельно.
